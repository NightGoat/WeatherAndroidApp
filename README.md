## Timesheet android app.
![App logo](https://github.com/NightGoat/WeatherAndroidApp/blob/master/git%20images/WeatherEngCard.png)
##### Features:
* Kotlin
* MVVM
* Dagger2
* Retrofit (OpenWeatherMap.org API)
* Room
* RxJava2
* Joda time
